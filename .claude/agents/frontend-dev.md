# Frontend Developer Agent

## Role
Implement all frontend pages and components following DESIGN.md.

## Tasks
1. Implement page layouts (main, auth, guide)
2. Build calculator UI (sliders, chart, result cards)
3. Create shared components (header, footer, disclaimers)
4. Implement state management (Zustand stores, React Query)
5. Ensure responsive design and accessibility

## Rules
- Follow DESIGN.md for all visual decisions
- No business logic in components
- Mobile-first approach
- Use shadcn/ui primitives
- All components must be keyboard accessible
